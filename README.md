Student Placement Data Analysis Project
 Overview

This project analyzes student placement data using **Python, SQL, and Power BI** to understand factors affecting student performance and placement readiness.
The goal is to derive meaningful insights from the dataset and present them through interactive dashboards.

---
 Tools & Technologies Used

*  Python (Pandas, Data Cleaning)
*  SQL (Data Analysis Queries)
*  Power BI (Dashboard Visualization)

---
Project Files

* `student_placement.ipynb` → Data cleaning and preprocessing using Python
* `student_placement.sql` → SQL queries for analysis
* `placement_readiness_dashboard.pbix` → Main dashboard (placement insights)
* `student_performance_dashboard.pbix` → Performance analysis
* `anxiety_dashboard.pbix` → Anxiety level analysis
* `math_students.pbix` → Math student performance analysis

---
 Key Insights

*  Students with **low anxiety** tend to score higher
*  **Study hours** positively impact total score
*  **Attendance percentage** plays a key role in performance
*  Placement readiness varies across **branches and grades**

---
 Dashboards Included

1. Student Placement Readiness Dashboard
2. Student Performance Dashboard
3. Anxiety Analysis Dashboard
4. Math Student Performance Dashboard

---
 How to Use

1. Open `.ipynb` file in Jupyter Notebook for Python analysis
2. Run `.sql` file in any SQL environment (MySQL / PostgreSQL)
3. Open `.pbix` files in Power BI Desktop to view dashboards

 Conclusion

This project demonstrates how data analysis techniques can be used to identify patterns in student performance and placement readiness, helping in better decision-making.
